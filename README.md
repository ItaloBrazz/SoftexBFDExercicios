## 🚀 Como Executar

1. **Clone o repositório**
   ```bash
   git clone https://github.com/italobrazz/softexbfdexercicios.git
   ```

2. **Acesse a pasta**
   ```bash
   cd aula*
   ```

3. **Instale as dependências (se necessário)**
   ```bash
   npm install
   ```

4. **Execute o arquivo desejado**
   ```bash
   node caminho/para/arquivo.js
   ```
